# 🐢 Turtle Soup AI · 海龟汤推理小游戏

一个基于 Spring Boot + 原生前端的海龟汤网页小游戏，  
引入大语言模型作为裁判，实现自然语言提问、胜利判定与完整游戏流程。

本项目为 Java 课程设计作品。

---

## 🎮 游戏玩法

- 系统随机给出一道「汤面」
- 玩家通过提问，根据 AI 回答的 **是 / 否 / 无关** 逐步推理真相
- 当你认为已经复盘出完整故事，可选择「我知道答案了」进行猜测
- AI 将基于语义理解进行胜利判定

---

## ✨ 功能特性

- 🤖 AI 裁判（是 / 否 / 无关）
- 📜 游戏日志记录
- 🔁 题目不重复轮换
- 🏁 胜利弹窗（WIN）
- 🍵 全部题目完成提示（END）
- 🔄 支持完整重置重新游玩
